**

Prerequisites
-----------
install node js - global   
install bower - global  
install gulp -global  
install mongodb   
**

How to start the application
----------------------------
In server folder -   
1.npm install  
2.mongod --dbpath PATH_OF_DATA --port 8086  
3.nodemon server /or node server/    

In public folder  
1.npm install  
2.bower install  
3.gulp build  

Access the application at http://localhost:3030/
